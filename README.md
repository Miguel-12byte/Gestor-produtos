📦 Gestor de Produtos

Aplicativo web progressivo (PWA) para gerenciamento de produtos com:

🚀 Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Remarcação de preços
- Leitura de código de barras
- Busca e filtros
- Funcionamento offline

📱 Instalação

1. Baixe o projeto
2. Abra o arquivo "index.html"

🔧 Tecnologias

- HTML
- CSS
- JavaScript

📦 APK

Pode ser convertido em APK usando Capacitor.
